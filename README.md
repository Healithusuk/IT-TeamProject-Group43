# README

### Requirements

This project is based on **Python 3.12.7**

Packages listed below should also be installed using **pip** before the application can run properly.

- Django 5.1.7
- bootstrap v5.3.0-alpha1
- mysqlclient 2.2.7
- pillow 11.1.0
- zxcvbn 4.5.0
